Introduction

Frequently, it is desirable to adjust the parameters within a single analysis, such as changing the dataset used for the analysis. Below, I will illustrate an example of how RMarkdown parametrisation allows for easy switching between different parameters. The used dataset is form European Centre for Disease Prevention and Control (ECDC) and can be found here. The dataset contains the reports of Covid-19 cases and deaths

Parameters

  • Parameter for country
  • Parameter for year
  • Parameter for month

Pseudocode

  • Load the dataset in
  • Select data for the parameters 
#read the Dataset sheet into ‚ÄúR‚Äù. The dataset will be called "data".

0.1.1 Data

This report has been written to represent the public data on COVID-19 cases and deaths, as published by the European Center for Disease Prevention and Control (ECDC) and accessible via: https://www.ecdc.europa.eu/en/publications-data/data-daily-new-cases-covid-19-eueea-country.

0.1.2 COVID-19 cases

The three graphs below depict the number of new COVID-19 cases as measured per day in the chosen month, the previous month and the following month.

data_plot %>% filter(month == params$month) %>% 
  ggplot(aes(x = day, y = cases)) +
  geom_line(size = .8, color = "orange") +
  labs(title = paste("COVID-19 cases in",country, month, year), x = "Day", y = "New cases")+
  scale_y_continuous(labels = comma, limits = c(0, max(data_plot$cases))) +
  scale_x_continuous(limits = c(1, 31)) +
  theme_bw()

Figure 4: Covid-19 cases in Netherlands March 2020

0.1.3 COVID-19 deaths

The three graphs below depict the amount of COVID-19 deaths as measured per day in the chosen month, the previous month and the following month.

data_plot %>% filter(month == params$month) %>% ggplot(aes(x = day, y = deaths)) +
  geom_line(size = .8, color = "red") +
  labs(title = paste("Covid-19 deaths in",country, month, year), x = "Day", y = "New deaths") +
  scale_y_continuous(labels = comma, limits = c(0, max(data_plot$deaths))) +
  scale_x_continuous(limits = c(1, 31)) +
  theme_bw()

Figure 5: Covid-19 deaths in Netherlands March 2020
