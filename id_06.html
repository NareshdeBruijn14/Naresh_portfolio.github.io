R package

I build an R package named {SynopsisPackage}. This package identify the data format and import the dataset if the correct data format is provided. After successfully importing the dataset, the package will provide a comprehensive summary that includes the following metadata:

  • Class: The datasets’ classification.
  • Data Type: The data type.
  • Dimensions: The datasets’ dimensions.
  • Structure: The datasets’ structure.
  • Column Names: The column names within the dataset.
  • Missing Values: Detailed information about missing values within the dataset.
  • Imported Dataset: The dataset itself, loaded into your environment.

    • By using this package, users can easily identify, read, and understand the basic structure and characteristics of their data. The package can be found in my github and can be installed and used with synopsis().

    Examples

    # install SynopsisPackage from github
if (!require("SynopsisPackage", character.only = TRUE)) {
  devtools::install_github("NareshdeBruijn14/SynopsisPackage", build_vignettes = TRUE)
  library(SynopsisPackage, character.only = TRUE)
}
