C.elegans plate experiment

Introduction

This exercise utilizes a dataset provided by J. Louter (INT/ILC) from an experiment involving adult C. elegans nematodes. The experiment aimed to investigate the effects of different compounds at varying concentrations on the reproductive output of these nematodes. Key variables in the dataset include the number of offspring (RawData), the generic name of the compound (compName), the concentration of the compound (compConcentration), and the type of experiment (expType). This dataset offers valuable insights into the dose-response relationships and the potential toxicological impacts of chemical exposure on C. elegans, serving as a model for broader biological and ecological studies.

Correction EtOH values

To calculate the concentration of ‚ÄúEthanol‚Äù in a solution given as a percentage and convert it to nano molar (nMol/L), follow the steps outlined below. The formula for converting the percentage of ethanol to nano molar (nMol/L) is as follows:

\[ \text{nMol/L} = \left( \frac{\text{Percentage EtOH} \times \text{Density in g/L}}{\text{Molar Mass in g/mol} \times 100} \right) \times 1.000.000 \]

Given resources:
  • 0.5% EtOH
  • 1.5% EtOH
Compound knowledge:
  • Density of EtOH: 789 g/L
  • Molar Mass: 46,068 g/mol

pseudocode

  • Convert percentage to fraction
  • Multiply by the density of ethanol
  • Divide by the molar mass of ethanol
  • Convert to nano molar 
perc_EtOH_to_nM <- function(percentage){
  
  perc_EtOH <- percentage
  density <- 789
  molar_mass <- 46.068
  
  # Convert percentage to fraction
  fraction_ethanol <- perc_EtOH / 100
  
  # Multiply by the density of ethanol
  concentration_g_per_L <- fraction_ethanol * density
  
  # Divide by the molar mass of ethanol
  concentration_mol_per_L <- concentration_g_per_L / molar_mass
  
  # Convert to nano molar
  concentration_nMol_per_L <- concentration_mol_per_L * 1e6
  
  # return result
  return(concentration_nMol_per_L)
  
}

Analysis plan

Pseudocode

Analysing Compound Concentrations:
  • Load the data (import dataset).
  • Filter the data to separate concentrations by compound (filter concentrations).
  • Create a bar graph representing the concentrations for each compound, with standard deviation bars as error bars (plotting).
  • Conduct a Shapiro-Wilk test separately for each group to assess normality (normality check).
  • Perform an analysis of variance (ANOVA) test to determine if there are significant differences between the compound groups (ANOVA test).
  • If the ANOVA test yields significance, proceed with post hoc tests to compare specific groups (post hoc test).
  • Draw conclusions based on the results obtained (conclusion).
Comparing LC50 Curves:
  • Load the LC50 data (import dataset).
  • Filter the data to obtain LC50 concentrations for each compound (filter concentration).
  • Plot the LC50 curves for each compound (plotting).
  • Calculate the 95% confidence intervals for each LC50 curve (95% confidence interval).
  • Assess the overlap between the 95% confidence intervals of different compounds (check overlap).
  • Draw conclusions based on the comparison of LC50 curves and the overlap of confidence intervals (conclusion).

Normalising CE_LIQ_FLOW_062_Tidydata

We are interested in how the number of nematode offspring changes after incubation with a substance compared to the normal situation. This information cannot be discerned from the above graph because the number of offspring in the negative control is not shown. To address this, I normalize the data by expressing the number of offspring in each measurement as a fraction of the average number of offspring after incubation in S-medium.

Conclusion

In the positive control group, significantly fewer nematodes were born after incubation in 1.5% ethanol compared to S-medium, as indicated by a fraction much smaller than 1. At substance concentrations below approximately 0.06 nM (see red dashed line or log10 values below -1.25), the number of nematodes born was either similar to, less than, or greater than the number born in the negative control group. Therefore, at these lower concentrations, there appears to be no noticeable effect on the number of offspring. However, at higher concentrations of the three tested substances, fewer nematodes were born compared to the negative control, indicating a significant impact on offspring production.
